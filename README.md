# COS 221 - Introduction to Databases

This repository houses all the practical assignments for the **COS 221 (Introduction to Databases)** module. 

## Structure

Each practical assignment is contained within its own respective folder.

- **`practical-4/`** - Practical 4: Database interactions and modeling using Java (JDBC) and MySQL.

## Prerequisites
To run the assignments in this repository, you may need:
- **Java Development Kit (JDK)**
- **Maven**
- **MySQL Server**
- **MySQL Workbench** (recommended for visually managing the databases and running SQL scripts)
